## Day 2 — Playwright Basics

**Topic:** Browser automation with Playwright (Python)  
**Tools:** Python 3, Playwright  
**Task:** Opens Wikipedia, searches for QA Automation, asserts page title, 
captures screenshot of results  

Install: `pip3 install playwright && playwright install`  
Run: `python3 search_test.py`

## What I learned
- Launching a real browser with Playwright
- Navigating to a URL
- Finding elements with locators
- Filling input fields and pressing Enter
- Asserting page title content
- Taking screenshots for visual evidence

## Bonus
Added screenshot capture — search_results.png saved automatically on each run.
This is used in real QA jobs for visual regression testing and failure documentation.
